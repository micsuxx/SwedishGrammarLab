What is a Consecutive Subjunction?

Consecutive subjunctions (konsekutiva subjunktioner) introduce subordinate clauses that express a result or consequence.

They show that something happens as a consequence of something else. In English, they often correspond to so ... that or so that.

Important

  • Consecutive subjunctions introduce result clauses.
  • They explain what something leads to.
  • Så ... att expresses degree + result.
  • Så att expresses result or purpose.
  • Så att ... inte expresses a negative result or prevented outcome.
Subjunktion Function Example Translations
Konsekutiva subjunktioner
så ... att Så ... att is used to express degree followed by result.

It corresponds to English so ... that.

The pattern is usually: så + adjective/adverb + att + subordinate clause.
Degree + Result
Hon var så trött att hon somnade mitt i filmen. 		Degree + Result
EN: She was so tired that she fell asleep in the middle of the film.
DE: Sie war so müde, dass sie mitten im Film einschlief.
FR: Elle était si fatiguée qu'elle s'est endormie au milieu du film.
IT: Era così stanca che si è addormentata a metà del film.
ES: Estaba tan cansada que se quedó dormida en medio de la película.
Degree + Result
Det var så kallt att sjön frös. 		Degree + Result
EN: It was so cold that the lake froze.
DE: Es war so kalt, dass der See zufror.
FR: Il faisait si froid que le lac a gelé.
IT: Faceva così freddo che il lago si è ghiacciato.
ES: Hacía tanto frío que el lago se congeló.
Degree + Result
Han körde så fort att polisen stoppade honom. 		Degree + Result
EN: He drove so fast that the police stopped him.
DE: Er fuhr so schnell, dass die Polizei ihn anhielt.
FR: Il conduisait si vite que la police l'a arrêté.
IT: Guidava così veloce che la polizia lo fermò.
ES: Conducía tan rápido que la policía lo detuvo.
så att Så att introduces a subordinate clause expressing result or purpose.

In spoken Swedish, att is often omitted in this construction.

The pattern is usually: så (att) + subject + verb.
Result / Purpose
Jag lämnade dörren öppen så att katten kunde gå ut. 		Result / Purpose
EN: I left the door open so that the cat could go out.
DE: Ich ließ die Tür offen, damit die Katze hinausgehen konnte.
FR: J'ai laissé la porte ouverte pour que le chat puisse sortir.
IT: Ho lasciato la porta aperta così che il gatto potesse uscire.
ES: Dejé la puerta abierta para que el gato pudiera salir.
Result / Purpose
Vi planerade resan noggrant så att inget skulle gå fel. 		Result / Purpose
EN: We planned the trip carefully so that nothing would go wrong.
DE: Wir planten die Reise sorgfältig, damit nichts schiefgehen würde.
FR: Nous avons soigneusement planifié le voyage afin que rien ne tourne mal.
IT: Abbiamo pianificato il viaggio con cura affinché nulla andasse storto.
ES: Planificamos el viaje cuidadosamente para que nada saliera mal.
Result / Purpose
Skriv tydligt så att läraren förstår vad du menar. 		Result / Purpose
EN: Write clearly so that the teacher understands what you mean.
DE: Schreib deutlich, damit der Lehrer versteht, was du meinst.
FR: Écris clairement pour que le professeur comprenne ce que tu veux dire.
IT: Scrivi chiaramente così che l'insegnante capisca cosa intendi.
ES: Escribe claramente para que el profesor entienda lo que quieres decir.
så att ... inte Så att ... inte expresses a negative result or prevented outcome.

It means that something is done so that something else does not happen.

The pattern is usually: så (att) + subject + inte + verb.
Negative Result
Han pratade tyst så att vi inte hörde vad han sa. 		Negative Result
EN: He spoke quietly so that we would not hear what he said.
DE: Er sprach leise, damit wir nicht hörten, was er sagte.
FR: Il parlait doucement pour que nous n'entendions pas ce qu'il disait.
IT: Parlava piano affinché noi non sentissimo ciò che diceva.
ES: Hablaba bajo para que no oyéramos lo que decía.
Prevented Outcome
Hon höll fast kopplet så att hunden inte sprang iväg. 		Prevented Outcome
EN: She held the leash tightly so that the dog would not run away.
DE: Sie hielt die Leine fest, damit der Hund nicht weglief.
FR: Elle tenait fermement la laisse pour que le chien ne s'enfuie pas.
IT: Teneva stretto il guinzaglio affinché il cane non scappasse.
ES: Sujetó bien la correa para que el perro no saliera corriendo.
Prevented Outcome
Vi stängde fönstret så att det inte skulle bli kallt. 		Prevented Outcome
EN: We closed the window so that it would not get cold.
DE: Wir schlossen das Fenster, damit es nicht kalt wurde.
FR: Nous avons fermé la fenêtre pour qu'il ne fasse pas froid.
IT: Abbiamo chiuso la finestra affinché non facesse freddo.
ES: Cerramos la ventana para que no hiciera frío.
